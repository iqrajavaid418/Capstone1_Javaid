Finance Tracker (Java CLI)

Overview

A simple command-line application to track financial transactions.
Users can add deposits, make payments, view a ledger, and search transactions.

Features

Add Deposit (positive amount)
Make Payment (negative amount)
View Ledger:
  * All transactions
  * Deposits only
  * Payments only
Reports:

  * Search by Vendor
  * Search by Description
  * Search by Amount
  * Previous Month filter

Advanced Custom Search (Bonus):

  * Filter by date range, vendor, description, and amount

--- Technologies

Java
File I/O (CSV storage)
OOP concepts

--- How to Run

1. Open project in IDE (IntelliJ recommended)
2. Run `Main.java`
3. Follow on-screen menu

--- Data Storage

Transactions are stored in a CSV file:


date|time|description|vendor|amount
